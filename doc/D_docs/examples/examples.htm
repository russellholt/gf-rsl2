The basics

DR_String x, y;
x.New();
y = x;


In detail, this process looks like:





	

		DR_String x,y;

			

			declaration of two String references (DR). They are "null" meaning
			they have no implementation.		
		
	

	

		x.New();

			

			create an implementation for x, that is. x now "points" to a DO_String.		
		
	

	

		y = x;

			

			invokes

			

			DR_String::operator =(const DR_String&)

			

			make y be x. Since both x and y are references, this means that y will reference
			what x references. y is now equivalent to x.
