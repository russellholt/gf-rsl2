class generation

The Foundation 2.0 - 2.2 tool rsl2c++ has been completely revamped. It is now based on hierarchical templates to such a degree as to allow it to generate, by switching template directories,

  • DObject / DRef C++ classes (rsl2D)
  • Java classes (rsl2Java)
  • HTML (rsl2html)
  • CORBA IDL (rsl2idl)
  • Foundation 2.2 Resources (rsl2resource)
  • ???

It is easy to create new templates to generate custom versions of the above in addition to things we haven't thought of, such as poetry.'

Usage

Each program - rsl2D, rsl2Java, rsl2html, etc. are actually shell scripts.
