Class recycler

extends D

manages object recycling. may create and destroy objects instead depending on circumstances. There should be an instance of this class for every class in the system. Some classes, such as the enumerators, are not recycled but destroyed due to implememntation issues.

Instance Variables

List freeList ;
list of free objects
List inUse ;
list of objects in use
String classname ;
class name that we are attached to

Methods

Object get ( );
gets an object, either from the free list or from spawn()
void recycle ( Object o );
puts an object on the free list or destroys it
Object spawn ( );
creates an object of the type we know about
