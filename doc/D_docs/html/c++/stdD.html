DO_ *safe_get()

this method is how operator->() is implemented; the purpose is to get the object, correctly typed, in a way that can be directly dereferenced witout checking. Because it is possible for the object to be null, safe_get() will create it by calling New(). This is the correct behavior 99% of the time. If it is not, use either const_get() or look in D_macros.h to find the actual code. It's possible that throwing an exception is more appropriate in some cases instead of creating a new object.
