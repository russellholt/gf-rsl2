DString

DString is

  • DO_String - the implementation
  • DR_String - the "smart pointer" reference to DO_String
  • Global operators - non-member operator functions for DR_Strings.

Public Member Functions

RWCString& val ( ) ;

Returns a reference to the internal string of the DO_String. This allows general RWCString methods to be called directly rather than relying on their implementation in DR_String. Because it returns a reference, it allows the internal RWCString object to be replaced as well.

RWCString& operator() ( ) ;

Shorthand for val().

RWCString const_val ( ) const;

Similar to val() and operator(), but returns a copy of the RWCString object internal to DO_String (returns by value rather than by reference).
