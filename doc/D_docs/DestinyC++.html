Destiny C++

The Destiny standard for C++ brings features and flexibility of a dynamic language to C++, including memory management (object pooling & recycling), generic collections, abstract composite objects, virtual data, class objects, streaming. These features allow C++ to work as seamlessly as possible with, yet be independent of, a dynamic system like RSL - so that they are as complementary as possible.
